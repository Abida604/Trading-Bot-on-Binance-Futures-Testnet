# Trading-Bot-on-Binance-Futures-Testnet
#Binance Futures Trading Bot (Testnet)

A Python-based trading bot that connects to Binance Futures Testnet and supports market and limit order execution via CLI.

# Features

- Connects to Binance Futures Testnet API
- Places Market Orders
- Places Limit Orders
- Input validation for safety
- Logging system for trade tracking
- Environment-based secure API handling
- CLI-based execution

#Tech Stack

- Python 3.8+
- python-binance
- python-dotenv
- CLI (argparse)
